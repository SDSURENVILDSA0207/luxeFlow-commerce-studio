# LuxeFlow Email Templates

This folder documents where campaign email templates are defined for preview and future send integrations.

## Source of truth

Templates currently live in:

- `src/modules/email/templates.ts`

Each template is represented as responsive HTML with metadata:

- `id`
- `name`
- `subject`
- `preheader`
- `description`
- `html`

## Available templates

1. Promotional Sale
2. Featured Collection
3. New Arrivals

## Preview

Use the admin UI at:

- `/admin/email-templates`

to preview each template in desktop and mobile frames.
